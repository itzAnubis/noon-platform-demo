# Noon (ن) - Women Empowerment Platform

**Noon (ن)** is an integrated web platform designed to empower Egyptian women economically and socially. This project serves as a graduation demo, showcasing a comprehensive ecosystem for female entrepreneurs, job seekers, and learners.

## 🌟 Overview

The platform is built with a focus on accessibility and community, providing a space where women can showcase their crafts, find remote work, and access educational resources. The interface is fully localized in Arabic (RTL) and features a modern, responsive design.

## 🚀 Key Features

- **Marketplace (سوق نُ):** A dedicated space for selling handmade products and crafts (e.g., embroidery, ceramics, natural skincare) created by Egyptian women.
- **Jobs Portal (فرص عمل):** Listings for remote work opportunities tailored for women looking for flexible employment.
- **Training & Education (تدريب):** Access to courses and training programs to enhance professional and creative skills.
- **Success Stories (قصص نجاح):** A section dedicated to sharing inspiring journeys of women within the community.
- **Community Support (دعم):** Integration of psychological and social support resources.

## 🛠️ Tech Stack

- **Frontend:** React 18 with TypeScript
- **Styling:** Tailwind CSS (Mobile-first, RTL support)
- **Icons:** Lucide React
- **Backend/Database:** Supabase (Ready for integration)
- **Build Tool:** Vite
- **Linting:** ESLint

## ⚙️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd noon_graduation
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🎓 Graduation Project Context
